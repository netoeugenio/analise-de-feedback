# analise-de-feedback
Este projeto implementa um modelo de Inteligência Artificial aplicado à Análise Preditiva de Sentimentos, utilizando técnicas de Processamento de Linguagem Natural (NLP) e Aprendizado de Máquina supervisionado.  O objetivo é classificar automaticamente textos de reviews como Positivos ou Negativos, transformando dados qualitativos em informação
